Q2. Number of operations

What are the number of operations for the following function?
public static void func(int n)
{
    int sum=0;
    for(int i=2;i<n;i+=2)
        sum+=i;
    System.out.println(sum);
}

Options
This problem has only one correct answer
(a) k1+k2(n) 
(b) k1+k2
(c) k1+k2(n^2)
(d) k1+k2(logn)

Ans. (a) k1+k2(n) ✓

Q4. Time complexity of a code

What will be the Time Complexity of following code in terms of ‘n’ ?
public static void func(int n)
{
    int sum=0;
    for(int i=1;i<n;i++)
        {
            for(;i<n*n;i++)
            {
                sum+=i;
            }
        }
    System.out.println(sum);
}

Options
This problem has only one correct answer
(a) O(n)
(b) O(n^2) 
(c) O(n^3)
(d) O(n^4)

Ans. (b) O(n^2) ✓

Q8. Time complexity of a code

{
    int sum=0;
    for(int i=1;i<n;i*=2)

        }
    System.out.println(sum);
}

(a) O(log n(base 2)) 
(b) O(n)
